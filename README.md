# 🎮 Koktel Tales

**Koktel Tales** is a wave-based, turn-based RPG built with **Rust** and the **Macroquad** game engine. Embark on an epic journey, master unique class abilities, conquer challenging waves, and face off against fearsome bosses!

---

## 🚀 Key Features

* **Dynamic Class System:** Play as a **Warrior**, **Rogue**, or **Mage**, each with unique playstyles and talent trees.
* **Boss Fight Mechanics:** Epic boss encounters every 5 waves with scaling difficulty and unique warning systems.
* **Talent Progression:** Deep customization options to fine-tune your character's strengths.
* **In-Game Shop:** Purchase powerful items and gear to boost your stats between battles.
* **Tactical Combat:** Utilize status effects like Freeze, Burn, and Poison to overcome elite enemies.
* **Immersive UI:** A classic, clean terminal-inspired interface with animated sprites.

---

## 🛠 Tech Stack

* **Language:** [Rust](https://www.rust-lang.org/)
* **Engine:** [Macroquad](https://macroquad.rs/)
* **Architecture:** State-driven game loop
